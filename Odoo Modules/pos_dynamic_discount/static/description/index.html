POS Dynamic Discount

This module enhances the discount functionality in the Odoo Point of Sale (POS) system. It introduces a more robust and user-friendly discount application feature, allowing users to apply global discounts on orders based on a selected type—either as a percentage or as a fixed amount.



Features
  • Discount Type Selection.
  • Percentage-based Discount: Applies a percentage discount to the total amount.
  • Fixed Amount Discount: Deducts a fixed monetary value from the total.
  • Support Community and Enterprise Editions
